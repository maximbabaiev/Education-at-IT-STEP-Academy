1) Получать список пользователей с jsonplaceholder и отображать в виде карточек
2) Добавить сортировку по нажатию на кнопки (сортировка по id и по имени)
3) По нажатию кнопки на карточке удалять пользователя и перерисовывать список
4) По нажатию на кнопку Add отображается форма в которую добавить валидацию полей (все поля обязательны, номер только цифры, проверка на пустые пробелы)
5) По нажатию на кнопку Create создать объект нового пользователя и добавить его в массив пользователей и перерисовать список карточек
